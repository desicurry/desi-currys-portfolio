About me

Coding is Life

Building websites is what I like to do the most. With a background in sales I decided to learn to code. I gained my knowledge at Lambda School where I learned to use HTML, CSS and JavaScript to create wonderful websites. Please feel free to check out some of my work.

My process is streamlined

Research everything

I gather all information need for the project. If it is something I need to understand I research and ask important questions before I start and project.

Design everything

Before I start coding I try to layout and design how the website should look and work and I set deadlines for completion.

Build it!

Here is where i start the process of building the website. I break the project down into sections and follow the design until it is complete.

Ship it like it’s hot

Once the project is complete I demo the website to make sure it is what was asked and making any changes as requested.

Show off my skills

I like to include and projects that I do in my portfolio to show the kind of work that I can do.

Give everyone a hi five 

I also work well in teams and as a team we can achieve much better things!
